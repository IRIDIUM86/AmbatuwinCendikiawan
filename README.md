# AmbatuwinCendikiawan

In order to use Suppabase, you have to install Supabase SDK first with the following command:
"npm install @supabase/supabase-js" for Frontend usage


For backend, due to pyiceberg cannot be installed with pip 3.14 somehow, we are going to use venv:
1. "py -3.13 -m venv venv"
2. ".\venv\Scripts\activate" must be run before doing anything to use py 3.13
3. "pip install supabase"


Bedrock installation:
"pip install boto3"


to access the front end, run these in terminal:
1. "npm install"
2. "npm start"
3. "npm run build"
