## 📚 What is Impeccable?

Impeccable is a comprehensive frontend design skill that helps you:

### Design & Improve UI
- Design, redesign, and shape frontend interfaces
- Critique and audit existing designs
- Polish, clarify, and distill UI elements
- Optimize and harden production interfaces

### Coverage Areas
- **Surfaces**: Websites, landing pages, dashboards, product UI, app shells, components, forms, settings, onboarding, empty states
- **Design Aspects**: UX review, visual hierarchy, information architecture, cognitive load
- **Technical**: Accessibility, performance, responsive behavior, theming
- **Visual**: Typography, fonts, spacing, layout, alignment, color, motion, micro-interactions
- **Content**: UX copy, error states, edge cases, i18n
- **Systems**: Reusable design systems and tokens

### Special Capabilities
- Make bland designs bolder and more delightful
- Make loud designs quieter and more refined
- Live browser iteration on UI elements
- Create ambitious visual effects that feel technically extraordinary

## 🎯 When to Use

Use the Impeccable skill when you need to:
- Improve the visual design of your frontend
- Audit UI/UX for best practices
- Create or refine design systems
- Optimize user experience
- Add animations and micro-interactions
- Ensure accessibility compliance
- Polish existing interfaces

## 🚀 How to Activate

To use the Impeccable skill in Kiro, you can:

1. **Automatic Activation**: When you ask about UI/UX design tasks, Kiro will automatically activate this skill
2. **Manual Activation**: Use the `discloseContext` tool with the skill name "impeccable"

Example tasks that will activate Impeccable:
- "Improve the design of this component"
- "Audit the UX of this page"
- "Make this interface more polished"
- "Add animations to this button"
- "Review the accessibility of this form"

## 📋 Setup Requirements

Before using Impeccable for design work, you should have:

1. **PRODUCT.md** (required) - Defines users, brand, tone, strategic principles
2. **DESIGN.md** (optional but recommended) - Defines colors, typography, elevation, components

These files help Impeccable understand your project's design context and brand guidelines.

## 🎨 Design Registers

Impeccable works with two design registers:

1. **Brand Register**: For marketing, landing pages, campaigns, long-form content, portfolios (design IS the product)
2. **Product Register**: For app UI, admin panels, dashboards, tools (design SERVES the product)
