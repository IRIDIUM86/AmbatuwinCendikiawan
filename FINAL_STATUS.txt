================================================================================
                            TROUBLESHOOTING
================================================================================

BACKEND WON'T START:
  $ pip install -r requirements.txt

FRONTEND WON'T START:
  $ npm install
  $ npm cache clean --force
  $ npm start

CORS ERRORS:
  Already fixed! Backend has CORS enabled

CAN'T CONNECT TO BACKEND:
  - Check backend is running on port 5000
  - Verify .env has REACT_APP_API_BASE_URL=http://localhost:5000/api
  - Restart frontend after checking .env

DATABASE ERRORS:
  - Verify Supabase credentials in .env
  - Check table name is bazaar_events
  - Verify supabaseUrl and supabaseKey

AWS BEDROCK ERRORS:
  - Verify AWS credentials in .env
  - Check AWS region is ap-southeast-5
  - Verify Bedrock model access in AWS console
